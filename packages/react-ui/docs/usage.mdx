## Installation

`react-ui` is published as a single package which can be installed via the npm registry.

```bash
npm install react-ui --save

# or if you prefer yarn

yarn add react-ui
```

<br />

### Built in components

`react-ui` comes with a set of in-built components to get you started. These components come with a default theme that is fully customisable.

<br />

<div className="example code-first">

```jsx
import { Form, Input, Button, ThemeProvider } from 'react-ui'

function App() {
  return (
    <ThemeProvider>
      <Form marginTop={10}>
        <Form.Header as="h2">Create an account</Form.Header>
        <Form.Field label="Full name">
          <Input placeholder="Enter your username" />
        </Form.Field>
        <Form.Field label="Email">
          <Input placeholder="Enter your email" />
        </Form.Field>
        <Form.Field label="Create a password">
          <Input placeholder="Enter a password" />
        </Form.Field>
        <Form.Field>
          <Button>Sign up</Button>
        </Form.Field>
      </Form>
    </ThemeProvider>
  )
}
```

### Customising the theme
  
The theme lets you control the overall design of your project with colors, fonts, sizes and spacing grid.

You can create a new theme from scratch or customise an existing one.

```js
import light from 'react-ui/themes/light' 

const theme = {
  ...light,
  fonts: {
    body: 'Inter, sans-serif',
    code: 'Dank Mono, monospace'
  }
}
```

Read more: [Theme specification](/docs/theming)

<br/>

### Customising built-in components

#### Approach 1:

You can change the design of any component from `react-ui` without overwriting styles in your application. You can do by passing a `components` prop the the `ThemeProvider` that looks like this:

```js
const components = {
  FormHeader: {
    textAlign: 'center'
  }
}

<ThemeProvider components={components}>
  <App/>
</ThemeProvider>
```


`react-ui` uses object styles that are hooked up to the theme, so you can use tokens from the theme.

```js
components = {
  FormHeader: {
    textAlign: 'center',
    color: 'greens.700'
  }
}
```

To find the reference key of a component, you can check the attached `data-component` on the element or find it in the [components index](docs/components/).

![data attribute](/docs/data-attr.png)

<br/>

#### Approach 2:

If you prefer to hide the implementation detail from approach 1, you can create a wrapper around a `react-ui` component that adds your styles to it. These styles are merged into the base styles.

```js
import React from 'react'
import { Form } from 'react-ui'

const extraStyles = {
  FormHeader: {
    textAlign: 'center',
    color: 'greens.700'
  }
}

const MyForm = props => {
  return (
    <Form css={extraStyles} {...props} />
  )
}

export default Form
```

### Customising the layout

When it comes to layout, `react-ui` has a few layout components to help you.

The responsibility of _laying_ out the elements falls on the container based on the context. That's why components from `react-ui` don't have any margin on them.

Instead, each component accepts [margin props](/docs/utilities/margins) that can be set at the point where they are used.

For example:

<div className="example code-first">

```jsx
<Element marginX={4} marginY={8}>
  <Heading marginBottom={4}>Secure your account</Heading>
  <Input placeholder="Enter a password" marginBottom={1} type="password" />
  <Text size={2} color="grays.700">
    Your password should contain 8 characters
  </Text>
</Element>
```

<ThemeProvider>
  <Element marginX={4} marginY={8}>
    <Heading marginBottom={4}>Secure your account</Heading>
    <Input placeholder="Enter a password" marginBottom={1} type="password" />
    <Text size={2} color="grays.700">Your password should contain 8 characters</Text>
  </Element>
</ThemeProvider>

</div>

The values of margins props are taken from the `space` scale in your theme.

<br/>

### One off use cases

There are rare occasions when you want to change a component but only for a single use case. Inline styles are the recommended way of doing so.

Components built with `react-ui` tap into the `style` prop and let you use theme tokens even in inline styles. 

Example: (You can inspect in developer tools to see the applied styles)

<div className="example code-first">

```jsx
<Element style={{backgroundColor: 'reds.300', padding: 4}}>
  <Heading marginBottom={4}>Secure your account</Heading>
  <Input placeholder="Enter a password" marginBottom={1} type="password" />
  <Text size={2}>
    Your password should contain 8 characters
  </Text>
</Element>
```

<ThemeProvider>
  <Element style={{backgroundColor: 'reds.300', padding: 4}}>
    <Heading marginBottom={4}>Secure your account</Heading>
    <Input placeholder="Enter a password" marginBottom={1} type="password" />
    <Text size={2}>
      Your password should contain 8 characters
    </Text>
  </Element>
</ThemeProvider>

</div>
